# Fuentes Requeridas

## Helvetica Now Display XBold

Para usar la fuente Helvetica Now Display XBold, necesitas:

1. **Comprar la licencia** de Helvetica Now Display en:
   - Monotype: https://www.monotype.com/fonts/helvetica-now
   - MyFonts: https://www.myfonts.com/fonts/monotype/helvetica-now

2. **Descargar los archivos** en formatos:
   - `HelveticaNowDisplay-XBold.woff2`
   - `HelveticaNowDisplay-XBold.woff`

3. **Colocarlos en esta carpeta** (`/fonts/`)

## Alternativa Gratuita

Mientras tanto, el sitio usa **Space Grotesk** como fallback, que tiene una estética similar moderna y geométrica.

Si prefieres usar una alternativa gratuita similar:
- **Sora** (Google Fonts) - https://fonts.google.com/specimen/Sora
- **Plus Jakarta Sans** (Google Fonts) - https://fonts.google.com/specimen/Plus+Jakarta+Sans
- **DM Sans** (Google Fonts) - https://fonts.google.com/specimen/DM+Sans
