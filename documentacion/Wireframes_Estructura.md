## 🔗 INTERACCIONES CLAVE

| Elemento | Interacción |
|----------|-------------|
| Video Hero | Autoplay silenciado, loop |
| Project Cards | Hover: reveal de info + zoom imagen |
| Service Cards | Hover: borde accent + slide de descripción |
| Navigation | Scroll: background blur aparece |
| CTAs | Hover: slide de color / scale |
| Gallery | Lightbox al click |
| Booking Calendar | Integración Calendly o similar |
