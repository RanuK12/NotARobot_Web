¿Qué es Webflow? Es una plataforma donde se construyen webs con un editor visual. La ventaja es que ustedes pueden editar textos y fotos sin tocar código. La desventaja es que tiene un coste mensual de hosting.

⚠️ Importante

Este presupuesto es para construir la web dentro de Webflow. Algunas cosas (como el cursor personalizado o ciertos efectos) pueden quedar más limitadas en Webflow comparado con código puro.

✓ Ventajas de Webflow:

  • Pueden editar textos e imágenes ustedes mismos
  • Panel visual fácil de usar (sin código)
  • Hosting incluido en la plataforma
  • Backups automáticos

✗ Costes adicionales de Webflow:

  • Plan CMS: ~$23/mes (~€21/mes) — Necesario para web con dominio propio
  • Coste anual: ~€250/año solo de hosting
  • Limitaciones: Algunos efectos avanzados (cursor, blur blobs) no son posibles o son más complicados
